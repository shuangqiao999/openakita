# install-store-skill

Install a Skill from the OpenAkita Platform Skill Store to the local system.

## Tools

- `install_store_skill` - Install a Skill from the platform
- `get_store_skill_detail` - View detailed info about a Skill before installing

## Usage

Use this skill when the user wants to:
- Install a Skill from the Skill Store
- Add new capabilities from the community
- View a Skill's details, trust level, and ratings before installing

## Parameters

### install_store_skill
- `skill_id` (required): The platform Skill ID to install

### get_store_skill_detail
- `skill_id` (required): The platform Skill ID to inspect

## Examples

- "安装 web-scraper Skill"
- "看看这个 Skill 的详情和评分"
- "从 Store 安装最受欢迎的数据分析 Skill"
