# search-store-skills

Search for Skills on the OpenAkita Platform Skill Store.

## Tools

- `search_store_skills` - Search and browse available Skills on the platform

## Usage

Use this skill when the user wants to:
- Find Skills on the OpenAkita Skill Store
- Browse available community and official Skills
- Discover Skills by trust level or category

## Parameters

- `query` (optional): Search keyword
- `category` (optional): Filter by category
- `trust_level` (optional): Filter by official, certified, or community
- `sort` (optional): Sort by installs, rating, newest, or stars
- `page` (optional): Page number for pagination
