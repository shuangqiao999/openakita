## 页面 1: 首页 - 任务列表 (Home Screen)

```
┌─────────────────────────────────────┐
│  ☰              TaskFlow       🔍   │  ← 顶部栏 (汉堡菜单 + 标题 + 搜索)
├─────────────────────────────────────┤
│  [全部] [未完成] [已完成] [标签▼]    │  ← 筛选标签栏 (可横向滑动)
├─────────────────────────────────────┤
│                                     │
│  ┌───────────────────────────────┐  │
│  │ ☐ 完成项目报告                 │  │
│  │    📅 今天 18:00  🔴 高优先级  │  │  ← 任务卡片 (未完成)
│  │    #工作 #紧急                  │  │
│  └───────────────────────────────┘  │
│                                     │
│  ┌───────────────────────────────┐  │
│  │ ☐ 准备周会材料                 │  │
│  │    📅 明天 10:00  🟡 中优先级  │  │  ← 任务卡片 (未完成)
│  │    #工作                        │  │
│  └───────────────────────────────┘  │
│                                     │
│  ┌───────────────────────────────┐  │
│  │ ✓ 购买办公用品                 │  │
│  │    📅 昨天     🟢 低优先级    │  │  ← 任务卡片 (已完成，灰色)
│  │    #生活                        │  │
│  └───────────────────────────────┘  │
│                                     │
│            ⋮                        │  ← 更多任务 (可滚动)
│                                     │
├─────────────────────────────────────┤
│                                     │  ← 底部留白区域
│                          ┌─────┐    │
│                          │  +  │    │  ← FAB 浮动按钮 (创建任务)
│                          └─────┘    │
└─────────────────────────────────────┘
```

**交互说明**:
- 点击复选框 (☐/✓): 切换任务完成状态
- 左滑卡片: 显示删除按钮
- 点击卡片: 进入任务详情页
- 下拉刷新: 更新任务列表
- 点击 FAB: 打开创建任务表单

## 页面 2: 创建/编辑任务表单 (Create/Edit Task)

```
┌─────────────────────────────────────┐
│  ← 取消          新任务       保存  │  ← 顶部栏
├─────────────────────────────────────┤
│                                     │
│  标题 *                             │
│  ┌───────────────────────────────┐  │
│  │ 输入任务标题...                │  │  ← 必填字段
│  └───────────────────────────────┘  │
│                                     │
│  描述                               │
│  ┌───────────────────────────────┐  │
│  │ 添加详细描述 (可选)            │  │
│  │                               │  │
│  │                               │  │
│  └───────────────────────────────┘  │
│                                     │
│  📅 截止时间                         │
│  ┌───────────────────────────────┐  │
│  │ 选择日期      >                │  │  ← 点击弹出日期选择器
│  └───────────────────────────────┘  │
│                                     │
│  🚩 优先级                           │
│  ┌─────────┐ ┌─────────┐ ┌─────────┐│
│  │  🔴 高  │ │  🟡 中  │ │  🟢 低  ││  ← 单选按钮组
│  └─────────┘ └─────────┘ └─────────┘│
│                                     │
│  🏷️ 标签                             │
│  ┌─────────┐ ┌─────────┐ ┌─────────┐│
│  │ #工作 + │ │ #生活 + │ │ #学习 + ││  ← 可选标签 (可自定义)
│  └─────────┘ └─────────┘ └─────────┘│
│                                     │
│            ⋮                        │
│                                     │
└─────────────────────────────────────┘
```

**交互说明**:
- 标题为空时，"保存"按钮置灰不可点击
- 点击日期字段：弹出系统日期选择器
- 点击优先级：单选，互斥
- 点击标签：多选，支持输入自定义标签
- 点击"取消"：二次确认"确定放弃编辑？"

## 页面 3: 任务详情页 (Task Detail)

```
┌─────────────────────────────────────┐
│  ← 返回                    ✏️  🗑️   │  ← 顶部栏 (编辑 + 删除)
├─────────────────────────────────────┤
│                                     │
│  ┌───────────────────────────────┐  │
│  │                               │  │
│  │   完成项目报告                 │  │  ← 任务标题 (大字号)
│  │                               │  │
│  │   📅 2026-04-07 18:00         │  │  ← 截止时间
│  │   🔴 高优先级                 │  │  ← 优先级标识
│  │                               │  │
│  └───────────────────────────────┘  │
│                                     │
│  ─────────────────────────────────  │  ← 分隔线
│                                     │
│  描述                               │
│  ┌───────────────────────────────┐  │
│  │ 需要完成 Q2 项目报告，包括:    │  │
│  │ 1. 数据分析                   │  │  ← 任务描述 (多行文本)
│  │ 2. 图表制作                   │  │
│  │ 3. PPT 汇总                    │  │
│  │                               │  │
│  └───────────────────────────────┘  │
│                                     │
│  ─────────────────────────────────  │
│                                     │
│  标签                               │
│  ┌─────────┐ ┌─────────┐           │
│  │ #工作   │ │ #紧急   │           │  ← 关联标签
│  └─────────┘ └─────────┘           │
│                                     │
│            ⋮                        │
│                                     │
│  ┌───────────────────────────────┐  │
│  │         标记为已完成           │  │  ← 底部操作按钮
│  └───────────────────────────────┘  │
│                                     │
└─────────────────────────────────────┘
```

**交互说明**:
- 点击右上角 ✏️: 进入编辑模式 (跳转页面 2)
- 点击右上角 🗑️: 弹出删除确认对话框
- 点击底部"标记为已完成": 切换状态，按钮变为"标记为未完成"
- 已完成任务：标题显示删除线，整体背景变灰

## 页面 4: 空状态 (Empty State)

```
┌─────────────────────────────────────┐
│  ☰              TaskFlow       🔍   │
├─────────────────────────────────────┤
│                                     │
│                                     │
│           📝                        │  ← 插图 (待办清单图标)

│                                     │

│                                     │
└─────────────────────────────────────┘
```

**使用场景**:
- 新用户首次打开应用
- 所有任务已完成/删除后

## 设计规范备注

### 颜色方案
| 元素 | 颜色值 | 用途 |
|-----|-------|------|
| 主色调 | #1976D2 (Material Blue) | FAB、按钮、选中状态 |
| 高优先级 | #D32F2F (Red) | 优先级标识 |
| 中优先级 | #FBC02D (Amber) | 优先级标识 |
| 低优先级 | #388E3C (Green) | 优先级标识 |
| 已完成 | #9E9E9E (Grey) | 文字、背景 |
| 背景色 | #FAFAFA | 页面背景 |
| 卡片背景 | #FFFFFF | 任务卡片 |

### 字体规范
| 元素 | 字号 | 字重 |
|-----|------|------|
| 页面标题 | 20sp | Bold |
| 任务标题 | 16sp | Medium |
| 任务描述 | 14sp | Regular |
| 辅助信息 | 12sp | Regular |

### 间距规范
- 卡片内边距：16dp
- 卡片间距：8dp
- 页面边距：16dp
- FAB 距离右下角：16dp
